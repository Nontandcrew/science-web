Electrochemistry

ไฟฟ้าเคมี(Electrochemistry)

คือ ความสัมพันธ์ระหว่างพลังงานไฟฟ้าและปฏิกิริยาเคมีซึ่งกระบวนการทางไฟฟ้าเคมีจะเกิดจากการถ่าย โอนอิเล็กตรอนจากสารหนึ่งไปสู่อีกสารหนึ่ง เมื่อมีการถ่ายโอนอิเล็กตรอน ปฏิกิริยาเคมีก็จะสามารถท าให้เกิด พลังงานไฟฟ้าได้ ในทางกลับกันพลังงานไฟฟ้าก็สามารถท าให้เกิดปฏิกิริยาเคมีได้เช่นกัน ปฏิกิริยาที่มีการถ่ายโอน อิเล็กตรอนนี้เรียกว่า ปฏิกิริยารีดอกซ์ (Redox Reaction) ซึ่งปฏิกิริยานี้สามารถสังเกตได้จากการเปลี่ยนแปลง ของเลขออกซิเดชัน (Oxidation Number) ของสารที่เกิดปฏิกิริยา

เลขออกซิเดชัน (Oxidation Number)

คือ ค่าประจุสมมติในแต่ละอะตอมในโมเลกุล โดยมีหลักการในการก าหนดเลขออกซิเดชันดังนี้
1. เลขออกซิเดชันของธาตุอิสระมีค่าเป็น 0 เช่น Na, O2 และ P4
2. โลหะหมู่ IA มีเลขออกซิเดชัน +1 และโลหะหมู่ IIA มีเลขออกซิเดชัน +2
3. H มีเลขออกซิเดชัน +1 ยกเว้นเมื่อเป็นสารประกอบ Hydride จะมีเลขออกซิเดชันเป็น -1
4. O มีโดยทั่วไปเลขออกซิเดชันเป็น -2 แต่จะมีเลขออกซิเดชันเป็น -1 และ -1/2 เมื่อเป็นสารประกอบ เปอร์ออกไซด์และซูเปอร์ออกไซด์
5. เลขออกซิเดชันของไอออนอะตอมเดี่ยวมีค่าเท่ากับประจุของไอออนนั้น
6. ผลรวมของเลขออกซิเดชันของสารที่เป็นกลางทางไฟฟ้ามีค่าเป็น 0

ปฏิกิริยารีดอกซ์ (Redox Reaction)

คือชื่อเรียกของปฏิกิริยารีดักชัน และ ออกซิเดชัน ซึ่งจะเกิดขึ้นควบคู่กันเสมอ
ปฏิกิริยารีดักชัน : เป็นปฏิกิริยาที่มีการให้อิเล็กตรอน สังเกตได้จากเลขออกซิเดชันที่เพิ่มขึ้นของอะตอม เช่น Zn A ( s ) ⟶ Zn A 2 + ( a q ) + 2 e A − ซึ่งจะเห็นว่าสังกะสีมีเลขออกซิเดชันเพิ่มจาก 0 เป็น +2
ปฏิกิริยาออกซิเดชัน : เป็นปฏิกิริยาที่มีการรับอิเล็กตรอน สังเกตได้จากเลขออกซิเดชันที่ลดลงของอะตอม เช่น Cu A 2 + A ( aq ) + 2 e − ⟶ Cu A ( s ) ซึ่งจะเห็นว่าทองแดงมีเลขออกซิเดชันลดจาก +2 เป็น 0
